Manuale Tecnico Generale del Sistema

Questo spazio raccoglie l'intera documentazione operativa per la gestione, lo sviluppo e la manutenzione di MoonOS. Il sistema integra il modulo ZRAM tramite zram-generator per la compressione della memoria in tempo reale ed esclude i colli di bottiglia dello swap tradicional su disco.

# =====================================================================
# 1. PROCESSO DI COMPILAZIONE DELLA ISO DAI SORGENTI
# =====================================================================

# Installazione dello strumento di generazione ufficiale Archiso
sudo pacman -S archiso

# Clonazione del repository delle configurazioni ed ingresso nella directory
git clone https://github.com/osmoon-developer/archiso-moonos-config.git
cd archiso-moonos-config

# Generazione dell'immagine ISO (work=file temporanei, out=destinazione ISO)
sudo mkarchiso -v -w ./work/ -o ./out/ ./nome_profilo/

# Pulizia profonda delle directory intermedie per liberare spazio su disco
sudo rm -rf ./work/


# =====================================================================
# 2. GESTIONE AMMINISTRATIVA DEI PACCHETTI (PACMAN E FLATPAK)
# =====================================================================

# Aggiornamento completo dei database remoti e dei pacchetti di sistema
sudo pacman -Syu

# Installazione di pacchetti software nativi dai repository
sudo pacman -S neovim tmux

# Rimozione pulita di un pacchetto eliminando le dipendenze orfane
sudo pacman -Rs nome_pacchetto

# Svuotamento della cache locale per recuperare spazio di archiviazione
sudo pacman -Sc

# Ricerca e installazione di applicazioni isolate in ambiente sandbox Flatpak
flatpak search firefox
flatpak install flathub org.mozilla.firefox

# Aggiornamento e pulizia dei runtime Flatpak inutilizzati dal sistema
flatpak update
flatpak uninstall --unused


# =====================================================================
# 3. DIAGNOSTICA DI MEMORIA E SCRITTURA IMMAGINE SU USB
# =====================================================================

# Verifica dello stato dei blocchi virtuali compressi ZRAM
zramctl

# Controllo complessivo della RAM e dello swap prioritario attivo
free -h

# Tracciamento dei blocchi hardware per individuare l'unità USB di destinazione
lsblk

# Scrittura diretta a basso livello (sostituire sdX con il nodo reale identificato)
sudo dd bs=4M if=moonos-2026.06-x86_64.iso of=/dev/sdX conv=fsync oflag=direct status=progress

# Svuotamento finale dei buffer prima dell'estrazione hardware
sync

Canali di Assistenza e Community

Per segnalare malfunzionamenti, coordinarsi nello sviluppo, richiedere l'aggiunta di pacchetti o ricevere supporto in tempo reale, la community si divide nei seguenti canali ufficiali centralizzati:

1. Segnalazione Bug e Tracciamento Malfunzionamenti (GitHub Issues)

Se riscontri errori o anomalie sistematiche nel sistema operativo, negli script o nelle build dei pacchetti, apri un bug report ufficiale all'interno della sezione dedicata del nostro repository centrale:

  • Centro Segnalazione Bug: github.com/osmoon-developer/issues/issues

2. Discussioni Liberi e Richieste di Supporto (GitHub Discussions)

Per fare domande generiche sulla compatibilità dell'hardware legacy, proporre idee per le prossime release o chiedere suggerimenti generali senza aprire una segnalazione di errore, utilizza il forum integrato:

  • Forum della Community: github.com/osmoon-developer/issues/discussions

3. Archivio dei Rilasci e Codici di Controllo

I file delle immagini stabili, i metadati e le relative somme di controllo hash (MD5, SHA256) per la validazione dei pacchetti scaricati sono pubblicamente consultabili qui:

  • Mirror dei file ISO su SourceForge: SourceForge Files
  • Repository dei codici hash di controllo: MoonOS_ISOs

4. Server di Comunicazione Istantanea (Discord)

Per la chat in tempo reale, il coordinamento rapido dei test ed il confronto diretto e immediato tra utenti e sviluppatori del progetto:

  • Server Discord Ufficiale: Accedi alla chat su Discord

MoonOS è un progetto open source gestito da osmoon-developer. Portale tecnico ospitato tramite GitHub Pages.
