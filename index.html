Sistemi datati, massime prestazioni.

Una distribuzione desktop basata sul cuore di Arch Linux e ottimizzata tramite i repository stabili di CachyOS. Progettata per essere leggera sulle macchine legacy ed estremamente reattiva su hardware moderno.

Scarica la ISO v2026.06

Architettura del Sistema

Kernel Altamente Ottimizzato

Sfrutta i moduli e lo scheduler del kernel CachyOS-Bore. Questo riduce la latenza dei processi in background, offrendo una fluidità superiore anche sotto carichi di lavoro intensi o su processori datati.

Gestione Avanzata della RAM

L'integrazione nativa di ZRAM tramite zram-generator permette al sistema di comprimere dinamicamente le pagine di memoria volatile. Riduce drasticamente l'uso dello spazio di swap su disco, prolungando la vita delle unità di archiviazione.

Pronto all'Uso

L'ambiente grafico viene configurato automaticamente durante l'installazione guidata con Calamares. Include il supporto preconfigurato a Flatpak e i tool personalizzati di gestione del sistema operativo.

Requisiti di Sistema e Dettagli

MoonOS è compatibile con l'architettura x86_64 standard, estendendo il supporto a CPU non recenti.

Componente Requisito Minimo Configurazione Consigliata
Processore Intel Pentium o equivalente x86_64 (v1) Intel Core / AMD Ryzen o superiori
Memoria RAM 2 GB (Grazie alle ottimizzazioni ZRAM) 4 GB o superiore
Spazio su Disco 15 GB liberi 30 GB su unità SSD
Interfaccia Grafica XFCE Desktop Environment personalizzato Gestore finestre con accelerazione attiva

Installazione e Verifica

Dopo aver completato il download dello specchio d'archivio ufficiale, puoi verificare l'integrità del file tramite il terminale locale. Questo assicura che l'immagine non sia corrotta.

sha256sum moonos-2026.06-x86_64.iso

Per i sistemi di sviluppo e il deployment automatizzato dell'ambiente di compilazione del profilo Archiso, i sorgenti ufficiali possono essere gestiti localmente tramite i seguenti passaggi controllati da Git:

git clone https://github.com/tuo-username/il-tuo-repo-moonos.git
cd il-tuo-repo-moonos
git checkout main
